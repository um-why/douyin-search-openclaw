# Skill Name: 抖音关键词搜索、竞品分析与热榜监控工具 (Douyin Search & Analytics)

## Description

抖音公开数据智能分析工具。支持**关键词搜索排序**、**抖人作品抓取**、**实时热榜获取**，适用于短视频营销、竞品分析和热点监控，助力爆款内容策划与流量追踪。
